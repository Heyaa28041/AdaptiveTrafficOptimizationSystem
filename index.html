About the Project

This system models each lane at an intersection using vehicle count, waiting time and priority. The urgency score drives a max heap selection and a multi-cycle optimization to minimize total delay and improve throughput.

  • Real-time urgency score computation
  • Priority queue scheduling with max heap
  • Greedy lane selection plus dynamic programming optimization
  • Adaptive green duration based on current traffic load
